WarpLocal 测试文档

单元测试 (unit/)

单元测试文件从 app/src/ 目录移动到这里，保持与源代码分离。

app_id_tests.rs

测试应用程序 ID 的生成和验证。

app_state_tests.rs

测试应用程序状态管理。

keyboard_tests.rs

测试键盘快捷键和输入处理。

input_suggestions_tests.rs

测试输入建议功能。

menu_tests.rs

测试菜单系统。

prefix_tests.rs

测试命令前缀处理。

preview_config_migration_tests.rs

测试预览配置迁移。

root_view_tests.rs

测试根视图组件。

safe_triangle_tests.rs

测试安全三角形算法。

集成测试 (integration/)

集成测试框架，用于测试多个组件之间的交互。

crates/integration/

自定义集成测试框架，包含测试工具和数据。
